# Exercise #
  
Given a sales report of different flights, find the N most sell routes. The file has this structure:  
`departure airport; arrival airport; sales`  

- - - - 
- - - - 

## Notes: ##

In the SQL world this means:

```sql
    select * 
    from TABLE 
    order by SALES desc 
    limit (N)
```
